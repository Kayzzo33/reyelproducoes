## Como Adicionar os Dados

1. Acesse `http://localhost:5173/admin`
2. Na aba "Projetos", clique em "Adicionar Projeto" e preencha com os dados acima
3. Na aba "Equipe", clique em "Adicionar Membro" e preencha com os dados reais da sua equipe
4. Substitua as URLs de exemplo pelos links reais das suas fotos e redes sociais

## Onde Hospedar Fotos

### Opção 1: Google Drive
1. Faça upload das fotos
2. Clique com botão direito > Obter link
3. Mude para "Qualquer pessoa com o link"
4. Use o link direto

### Opção 2: Imgur
1. Acesse [imgur.com](https://imgur.com/)
2. Faça upload da imagem
3. Copie o link direto

### Opção 3: Pexels (apenas temporário)
- Use fotos do Pexels apenas para teste
- Substitua por fotos reais o mais rápido possível

## Observações Importantes

- Todas as URLs devem começar com `https://`
- Os links do Instagram devem estar no formato: `https://instagram.com/username`
- Os links do Drive devem apontar para pastas públicas
- Recomenda-se usar fotos próprias para melhor representar o trabalho da equipe
